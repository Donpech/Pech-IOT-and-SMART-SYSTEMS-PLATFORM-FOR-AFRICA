## Document Standards

All documents follow these standards:

- **Header:** `PECH GROUP HOLDINGS LTD` with document title and confidentiality marking
- **Company Info:** Lagos, Nigeria | pechgroupholdings.tech
- **Reference Numbers:** Standardized format `[TYPE]-[YYYY]-[NNNN]`
- **Currency:** Nigerian Naira (₦)
- **Tax:** VAT at 7.5% (FIRS-compliant)
- **Legal Context:** Nigerian law, Lagos jurisdiction
- **Signatures:** Multi-level approval chains with Name, Signature, Date fields
- **Format:** Markdown tables for form fields, numbered clauses for terms

1. Copy the relevant template
2. Fill in company-specific details (bank accounts, TIN, RC number, etc.)
3. Convert to PDF, Word, or print format as needed
4. Ensure all reference numbers follow the sequential numbering system
5. Obtain required signatures per the approval chain
