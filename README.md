## How to Use These Templates

### 1. Select the Right Document

| Scenario | Document to Use |
|----------|----------------|
| Hiring a permanent employee | Full-Time Employment Contract |
| Engaging a freelancer or consultant | Independent Contractor Agreement |
| Bringing on an intern | Internship Agreement |
| Hiring a commission-only sales agent | Commission Marketer Agreement |
| Hiring an office-based marketer (salary + commission) | Office Marketer Employment Agreement |
| Engaging an API/platform developer | API Developer Agreement |
| Certifying an installer/technician | Certified Installer Agreement |
| Anyone who needs to sign an NDA | Non-Disclosure Agreement |
| All personnel (media conduct) | Media & Public Communications Policy |
| Running a hackathon event | Hackathon Rules & Guidelines |
| Requiring a guarantor for any hire | Guarantor Form |

### 2. Fill in the Blanks

All templates use `_______________` as placeholders. Fill in:
- **Names** — Full legal names of both parties
- **Dates** — Effective date, commencement date
- **Addresses** — Registered addresses
- **Contract Reference Number** — Use the format shown (e.g., `PECH/EMP/001/2026`)
- **Role-specific details** — Job title, salary, department, reporting line
- **Schedules** — Attach relevant schedules (job description, SOW, KPIs, etc.)

### 3. Reference Number Format

Each document type has its own prefix:

| Prefix | Document |
|--------|----------|
| `PECH/EMP/___/______` | Full-Time Employment |
| `PECH/CON/___/______` | Independent Contractor |
| `PECH/INT/___/______` | Internship |
| `PECH/NDA/___/______` | Non-Disclosure Agreement |
| `PECH/API/___/______` | API Developer |
| `PECH/INS/___/______` | Certified Installer |
| `PECH/CMA/___/______` | Commission Marketer |
| `PECH/OMA/___/______` | Office Marketer |
| `PECH/GUA/___/______` | Guarantor Form |
| `PECH/HCK/___/______` | Hackathon |

### 4. Export to Word/PDF

These Markdown files can be converted using:
- **Pandoc:** `pandoc contracts/PECH_FULL_TIME_EMPLOYMENT_CONTRACT.md -o contract.docx`
- **VS Code:** Use a Markdown-to-PDF extension
- **Online:** Paste into Google Docs or use a Markdown-to-PDF converter
- **Print:** Render in any Markdown viewer and print to PDF

### 5. Customize for Your Needs

- Adjust salary amounts, commission rates, and bonus thresholds
- Modify probation periods and notice periods within legal minimums
- Add or remove schedules as needed
- Update company address if it changes
- Always have a legal professional review before use

## Nigerian Legal Compliance

All documents are designed to comply with the following Nigerian legislation:

| Law | Key Provisions |
|-----|---------------|
| **Labour Act (Cap L1, 2004)** | Written contracts required for engagements >3 months. Minimum notice periods, leave entitlements. |
| **Pension Reform Act 2014** | Employer 10% + Employee 8% of monthly emolument for pension contributions. |
| **NHIA Act 2022** | Mandatory health insurance for organizations with 5+ employees. |
| **Group Life Insurance** | Mandatory 3x annual salary group life cover for employees. |
| **Employee Compensation Act 2010** | 1% of payroll contribution to NSITF. |
| **NDPA 2023** | Nigeria Data Protection Act — consent required for processing personal data. |
| **FIRS (Withholding Tax)** | 5% WHT on contractor payments; 10% on professional services. |
| **FCCPA 2019** | Non-compete clauses must be reasonable in scope, geography, and duration (max 2 years). |
| **Consumer Protection Act** | Compliance required for all customer-facing marketing and sales activities. |
| **Copyright Act** | Copyright defaults to employee unless explicitly assigned — IP assignment clauses are critical. |

> **Disclaimer:** These templates are provided for guidance only and do not constitute legal advice. Always consult a qualified Nigerian legal practitioner before executing any agreement.

| **NHIA** | Yes | No | No | Yes | No |
| **Group Life** | Yes (3x salary) | No | No | Yes (3x salary) | No |
| **Leave** | 20 days annual | Per contract | N/A | 20 days annual | As agreed |
| **IP Assignment** | Yes | Yes | Yes (leads/data) | Yes | Yes |
| **Non-Compete** | 24 months | 12 months | 12 months | 12 months | 6 months |
| **Termination Notice** | 1-3 months | 14 days | 14 days | 1-2 months | 7 days |
