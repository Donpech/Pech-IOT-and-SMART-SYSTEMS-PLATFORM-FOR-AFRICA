## 2. LOGO USAGE

### Primary Logo
- Hexagonal icon shape with blue-to-orange gradient (top to bottom)
- "PECH" wordmark alongside or below the icon
- The hexagonal shape represents technology, connectivity, and structure

### Logo Versions

| Version | Usage |
|---|---|
| **Full Color (Primary)** | Default use on white/light backgrounds |
| **White (Reversed)** | On dark backgrounds, photography overlays |
| **Single Color (Blue)** | Monochrome printing, embossing |
| **Single Color (Black)** | Black & white documents, fax |

### Clear Space
- Minimum clear space around the logo equals the height of the "P" in PECH
- No other text, graphics, or elements may intrude into the clear space

### Minimum Size
| Application | Minimum Width |
|---|---|
| **Print** | 25mm |
| **Digital** | 80px |
| **Favicon** | 32px (icon only, no text) |

### Logo Don'ts
1. Do NOT rotate or tilt the logo
2. Do NOT stretch or distort proportions
3. Do NOT change the gradient direction
4. Do NOT place on busy/cluttered backgrounds without sufficient contrast
5. Do NOT add drop shadows, outlines, or effects
6. Do NOT rearrange logo elements
7. Do NOT use unapproved color combinations
8. Do NOT crop or partially display the logo

### Secondary Colors

|---|---|---|---|
| **Dark Blue** | #0099CC | 0, 153, 204 | Darker shade for depth, subheadings |
| **Dark Orange** | #E08A00 | 224, 138, 0 | Darker shade for hover states, emphasis |

### Neutral Colors

| Color | Hex | RGB | Usage |
|---|---|---|---|
| **Dark Background** | #1B2838 | 27, 40, 56 | Dark navy backgrounds, headers |
| **White** | #FFFFFF | 255, 255, 255 | Text on dark, clean backgrounds |
| **Text Dark** | #333333 | 51, 51, 51 | Primary body text |
| **Text Medium** | #666666 | 102, 102, 102 | Secondary text, captions |
| **Text Light** | #999999 | 153, 153, 153 | Disclaimers, footnotes |
| **Light Gray** | #F5F5F5 | 245, 245, 245 | Subtle backgrounds |
| **Border Gray** | #E0E0E0 | 224, 224, 224 | Borders, dividers |

### Brand Gradient
- **Direction:** Top to bottom (or left to right)
- **Start:** PECH Sky Blue (#00BFFF)
- **End:** PECH Orange (#F5A623)
- Use sparingly for accent bars, decorative elements, and special features

### Color Usage Rules
1. Sky Blue is the dominant brand color — use it for primary headings and key visual elements
2. Orange is the accent color — use it for calls-to-action, highlights, and to draw attention
3. Never use Orange as a background for large areas
4. Maintain sufficient contrast ratio (minimum 4.5:1) for text readability
5. The gradient should only be used for decorative elements, never for text backgrounds

### Primary Typeface — Montserrat

| Weight | Usage |
|---|---|
| **Montserrat Bold** | Headlines, titles, primary headings |
| **Montserrat SemiBold** | Subheadings, emphasized text |
| **Montserrat Medium** | Navigation, buttons, labels |

| Weight | Usage |
|---|---|

| **Open Sans SemiBold** | Inline emphasis, table headers |

```

font-family: 'Open Sans', 'Helvetica Neue', Helvetica, Arial, sans-serif;
```

## 5. IMAGERY STYLE

### Photography Guidelines
- **Modern and clean** — Avoid cluttered or low-quality images
- **African-focused** — Feature African people, landscapes, and settings
- **Technology themes** — IoT devices, solar panels, smart systems, data visualizations
- **Warm lighting** — Align with brand's optimistic and warm tone
- **Authentic** — Use real scenarios over staged stock photos where possible

### Image Treatment
- Apply a subtle Sky Blue or Dark overlay (#1B2838 at 60% opacity) for text overlay images
- Round corners (8px radius) for inline images in presentations/web
- Use brand gradient as a border or accent on featured images

### Iconography
- Style: Line icons with 2px stroke weight
- Primary color: Sky Blue (#00BFFF)
- Secondary color: Orange (#F5A623) for active/selected states
- Maintain consistent icon size within each context

## 6. TONE OF VOICE

| Attribute | Description | Example |
|---|---|---|
| **Professional** | Competent and knowledgeable, but not stiff | "Our IoT solutions deliver measurable results for businesses across Africa." |
| **Warm** | Friendly and approachable, people-first | "We're here to help you power your future." |
| **Innovative** | Forward-thinking, solution-oriented | "Reimagining how Africa connects with smart technology." |
| **Trustworthy** | Honest, transparent, reliable | "Quality you can count on — that's the PECH promise." |

### Writing Guidelines
1. Use active voice over passive voice
2. Write in clear, simple language — avoid unnecessary jargon
3. When using technical terms, provide brief explanations
4. Be specific and data-driven where possible
5. Address the reader directly ("you" not "one" or "the customer")
6. Keep sentences concise — aim for 15-20 words per sentence

## 8. INCORRECT USAGE EXAMPLES

| Violation | Why It's Wrong |
|---|---|
| Using red or green as brand colors | Not in the brand palette |
| Placing logo on a busy photograph without overlay | Poor visibility and contrast |
| Using Comic Sans or decorative fonts | Does not align with professional brand image |
| Stretching the logo to fit a space | Distorts brand identity |
| Using Orange as a large background color | Orange is an accent color only |
| Mixing brand blue with different shades of blue | Creates inconsistency |
| Using low-resolution logo files | Appears unprofessional |
| Adding shadows or 3D effects to the logo | Violates logo integrity |
