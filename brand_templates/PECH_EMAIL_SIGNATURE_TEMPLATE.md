### Setup Instructions

#### Gmail
1. Go to **Settings** > **See all settings** > **General** tab
2. Scroll to **Signature** section
3. Click **Create new** and name it "PECH Official"
4. Paste the HTML code (use browser extension or copy from rendered preview)
5. Replace all placeholder values with your actual details
6. Click **Save Changes**

#### Outlook (Desktop)
1. Go to **File** > **Options** > **Mail** > **Signatures**
2. Click **New**, name it "PECH Official"
3. Paste the HTML signature
4. Set as default for new messages and replies
5. Click **OK**

#### Outlook (Web)
1. Go to **Settings** (gear icon) > **View all Outlook settings**
2. Go to **Mail** > **Compose and reply**
3. Paste signature in the editor
4. Set as default, click **Save**
