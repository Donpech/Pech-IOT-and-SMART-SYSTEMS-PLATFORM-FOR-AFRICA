### Logo Placement Rules

| Rule | Specification |
|---|---|
| **Position:** | Top-left corner (default) or top-right for special layouts |
| **Minimum Size:** | 30px height on content slides |
| **Maximum Size:** | 120px on title/closing slides |
| **Clear Space:** | Minimum 10px padding on all sides |
| **On Dark Backgrounds:** | Use white/light version of logo |
| **On Light Backgrounds:** | Use full-color version of logo |

---

### Slide Transitions & Animations

| Element | Animation |
|---|---|
| **Slide Transition:** | Fade, 0.5s duration |
| **Title Entry:** | Fade in from bottom, 0.3s |
| **Bullet Points:** | Appear on click, fade in, 0.2s each |
| **Charts/Images:** | Fade in, 0.4s |
| **Avoid:** | Spinning, bouncing, fly-in, or flashy animations |
