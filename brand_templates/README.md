## Overview

This directory contains **12 comprehensive brand identity templates** for PECH Group Holdings Ltd. These templates define the visual identity, design specifications, and usage guidelines for all branded materials.

## Brand Colors

| Color | Hex | Usage |
|---|---|---|
| **PECH Sky Blue** | `#00BFFF` | Primary brand color, headings, key elements |
| **PECH Orange/Amber** | `#F5A623` | Secondary brand color, accents, highlights |
| **Dark Blue** | `#0099CC` | Darker shade of brand blue |
| **Dark Orange** | `#E08A00` | Darker shade of brand orange |
| **Dark Background** | `#1B2838` | Dark navy for backgrounds |
| **White** | `#FFFFFF` | Text on dark backgrounds |

## Typography

| Typeface | Usage |
|---|---|
| **Montserrat Bold** | Headlines, titles, primary headings |
| **Open Sans Regular** | Body text, descriptions, form fields |
| **Arial/Helvetica** | Email-safe fallback |

---

## Usage Notes

1. **Start with Brand Guidelines** — Read `PECH_BRAND_GUIDELINES.md` first for overall brand rules
2. **Color Consistency** — Always use the exact hex codes listed above
3. **Logo Files** — Obtain approved logo files from the Marketing Department
4. **Approval** — All branded materials must be reviewed by Marketing before production
5. **Printing** — Use Pantone-matched colors for professional print materials
